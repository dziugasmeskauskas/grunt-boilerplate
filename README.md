# Gulp boilerplate
- SASS
- ES6 (Babel)
- Uglify JavaScript
- Browsersync
- Minify CSS
- Delete not used CSS rules
- Autoprefix CSS
- Image optimization

# Set up

## Requirements 

- Gulp `npm install gulp -g`

## Instalation 

- `yarn install`
- `gulp` for development
- `gulp build` for distribution
